Github Account

Create a Github account and provide us with the user name:

  1. Create a free account at Github, if you do not have one already.

  2. Once created, fill in this form before Wednesday Jan 27 Noon.

  3. Look out for an invitation email from Github in the mailbox of the email address you used to sign up for Github. Accept the invitation to join the course Github organization.

caution

Do not submit the above form more than once!

caution

If you do not accept the invitation, you will not be able to access the starter codes and we will not be able to provision you a private repositoty that you will need for the remainder of the semester. Once you accept the invitation, you will be given a private repo under jhu-ip organization which you will use as your personal space for the exercises and homeworks throughout the semester.
